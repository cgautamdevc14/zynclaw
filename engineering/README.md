# Engineering

Engineering owns the "how" and the code-level execution contract.

Use this folder to convert project-ready work into safe local-agent tasks:

- `AGENT_CONTRACT.md`: what every agent task must include.
- `EXECUTION_PLAYBOOK.md`: how local agents should work through a task.
- `CODE_REVIEW.md`: review checklist for agent-generated changes.
- `ADR_TEMPLATE.md`: architecture decision record template.

## Engineering Rules

- Prefer small, testable changes.
- Name files and commands up front.
- Keep local agents on execution, not ambiguous planning.
- Human review remains mandatory.
